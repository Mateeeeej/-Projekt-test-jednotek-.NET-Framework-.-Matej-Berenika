# Projekt testů jednotek (.NET Framework) - Cech dobrodruhů

**Autoři:** Matěj a Berenika

## Volitelná část: Jak by se dal systém rozšířit?
* Místo pouhého čísla u předmětů by se dal přidat plnohodnotný **Inventář**.
* Mohla by se přidat mechanika peněz (zlaťáky) pro obchodování s obchodníkem.
* Třída `ClenCechu` by měla být ideálně `abstract`, protože v samotné hře nebudeme chtít vytvářet jen tak bezejmenného "člena", ale vždycky buď Dobrodruha, nebo Obchodníka.
